# Roadmap

## MVP (Mostly Complete)
- Responsive shell, public pages, credential auth, protected/admin routing
- CRUD module with API persistence
- Admin users, settings, and audit-log endpoints with UI wiring
- AI assistant page with persisted message history

## v1 (In Progress)
- TanStack Router / TanStack Start migration
- Security hardening: CSRF protection and production session middleware refinements
- Expand DB-backed business logic for deeper dashboard and user analytics
- Broaden integration and e2e coverage

## v2
- Team/workspaces, billing, notifications, and plugin modules
- Multi-tenant patterns and advanced admin controls
- Agentic workflow modules and extensibility framework
